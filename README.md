# zenn-trend-cli

📝 A command to fetch [Zenn](https://zenn.dev) trend

## Usage

```
zenn-trend-cli [-h|--help] [-V|--version]
```

## Installation

```
deno install --allow-net --name zenn-trend-cli \
'https://raw.githubusercontent.com/sheepla/zenn-trend-cli/master/cli.ts'
```
